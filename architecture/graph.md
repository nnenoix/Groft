# Dependency Graph

Граф зависимостей между задачами и модулями. Показывает, что от чего зависит, какие задачи можно выполнять параллельно, а какие строго последовательно. Обновляется тимлидом при планировании новых задач и при обнаружении новых зависимостей.

---

## HEALTH-1

```
HEALTH-1 (Express GET /health)
  ├─ backend-dev: реализация server.js + package.json
  └─ tester: smoke test запуска сервера и запроса /health
```

Зависимостей от других задач нет — это первая задача в проекте. Frontend не участвует.
