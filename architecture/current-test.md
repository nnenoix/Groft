# Текущий тест

Сюда Opus записывает тест для задачи ПЕРЕД её выдачей исполнителю — в рамках TDD-цикла. Исполнитель получает задачу вместе с этим тестом и пишет код до тех пор, пока тест не начнёт проходить. Tester запускает тест и возвращает результат тимлиду.

Формат записи: ссылка на файл теста в репозитории + короткое описание что проверяет тест + команда запуска. Тест должен быть исполняемым (не псевдокодом).
